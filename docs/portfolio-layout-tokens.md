# Portfolio Layout Tokens

## Spacing
- Page horizontal padding: `1.25rem` mobile, `2rem` tablet and above
- Hero vertical padding: `5.5rem` mobile, `7.5rem` desktop
- Section vertical padding: `4.5rem` mobile, `5.5rem` desktop
- Contact vertical padding: `5.5rem` mobile, `7.5rem` desktop
- Card padding: `1.75rem`
- Inline chip padding: `0.5rem 1rem`

## Typography
- Display: `3.5rem`, line-height `1.08`, weight `800`
- Headline: `2.5rem`, line-height `1.16`, weight `800`
- Title: `1.5rem`, line-height `1.34`, weight `700`
- Body: `1.0625rem`, line-height `1.7`, weight `500`
- Caption: `0.875rem`, line-height `1.5`, weight `600`

## Motion
- Section reveal: `0.55s`, ease `cubic-bezier(0.2, 0, 0, 1)`, translateY `1.75rem`
- Work card hover: scale `1.015`, translateY `-0.25rem`, duration `0.22s`
- Button hover: duration `0.2s`, same easing
